# 美伊战争局势看板 - 设计文档

## 项目目标

构建一个单页看板，聚合美伊战争相关的核心市场指标，支持手动刷新获取最新数据，展示趋势变化，
通过四层因果分析框架给出经济体制判定、情景概率、操作建议和拐点监控。

## 技术栈

- **前端**: 单页 HTML + CSS + JavaScript（无框架，轻量）
- **后端**: Python FastAPI
- **数据存储**: SQLite（存储历史数据用于趋势图）
- **图表**: Chart.js（轻量级前端图表库）

## 分析框架：四层因果链

核心思路：指标不是孤立的，它们形成一条因果传导链。

```
第1层：冲突走向判断（领先指标）
         ↓
第2层：供给冲击传导（实时指标）
         ↓
第3层：经济体制判定（组合指标）
         ↓
第4层：情景推演与操作建议（输出）
```

### 第1层：冲突走向判断（领先指标）

这些指标具有预测性质，是最有价值的前瞻信号：

| 指标 | 为什么领先 |
|------|-----------|
| Polymarket 停战概率 | 最直接的前瞻指标，聚合全球信息的市场定价 |
| 伊朗里亚尔汇率 | 地面真相——汇率崩到什么程度，伊朗才有谈判意愿 |
| 军工股涨幅 | 聪明钱判断——持续创新高=资金押注冲突长期化 |
| VIX 水平 | 市场整体恐慌度，但也可能是情绪过度反应 |

**拐点信号组合**：里亚尔加速贬值 + 停战概率从底部回升 + 军工股见顶回落 → 冲突拐点。

### 第2层：供给冲击传导（实时指标）

冲突通过两条路径影响全球经济：

```
霍尔木兹海峡封锁
  ├── 原油 (Brent/WTI) ──→ 汽油/柴油 ──→ 运输成本 ──→ 所有商品
  ├── 天然气 (LNG) ──→ 电力/化工成本
  └── 航运中断 ──→ 运价暴涨 ──→ 全球贸易摩擦成本

能源成本上升
  ├── 化肥涨价 ──→ 小麦/玉米上涨 ──→ 全球粮食通胀
  ├── 铝冶炼成本 ──→ 工业品涨价
  └── 通胀预期 (TIP/RINF) ──→ 央行政策被困
```

### 第3层：经济体制判定

多指标组合判定当前处于哪种经济体制，这决定了操作方向：

| 组合信号 | 体制 | 含义 |
|---------|------|------|
| 油价↑ + 美股↓ + 国债收益率↑ + 黄金↑ | **滞胀** | 最危险：增长停滞+通胀，央行无法救市 |
| 油价↑ + 美股↓ + 国债收益率↓ + 黄金↑ | **衰退恐慌** | 市场预期经济硬着陆，资金涌入国债避险 |
| 铜价↓ + 美股↓ + 印度VIX↑ | **全球衰退** | 工业需求崩塌，新兴市场首当其冲 |
| 美元↑ + 美股↓ + 新兴市场↓ | **美元虹吸** | 资本从全球撤回美国，新兴市场承压 |
| 油价↓ + VIX↓ + 美股↑ + 黄金平 | **正常化** | 冲突缓和，市场回归基本面 |

判定规则（代码实现）：
- 滞胀：brent涨>5% 且 sp500跌>1% 且 us10y涨>0 且 gold涨>0
- 衰退恐慌：brent涨>5% 且 sp500跌>1% 且 us10y跌 且 gold涨>0
- 全球衰退：copper跌>2% 且 sp500跌>1% 且 india_vix>20
- 美元虹吸：usd涨>0.5% 且 sp500跌>1% 且 (ksa跌 或 uae跌)
- 正常化：vix<20 且 brent<80 且 sp500涨

### 第4层：三种情景与操作建议

#### 情景概率判定

基于 Polymarket + 军工 + 里亚尔综合判断：

| 情景 | 基准概率 | 修正条件 |
|------|---------|---------|
| A. 快速停火 (1-2月内) | Polymarket最近deadline的概率 | 里亚尔加速贬值→上调；军工创新高→下调 |
| B. 持久空袭 (3-6月) | 100% - A - C | 中间状态 |
| C. 全面升级 (地面战/扩大化) | 基准15% | 沙特ETF暴跌→上调；美股暴跌>5%单日→上调 |

#### 操作建议矩阵

**确定性较高（不依赖情景）：**

| 方向 | 标的示例 | 逻辑 |
|------|---------|------|
| 做多黄金 | 黄金ETF(518880)、AU9999、GLD | 三种情景下黄金都不差：避险+通胀对冲 |
| 做多能源 | 原油ETF、中国石油/中海油、XLE | 霍尔木兹哪怕部分恢复，供应缺口短期难补 |
| 做多军工 | 军工ETF(512660)、LMT/RTX/NOC | 全球军费开支结构性上升 |
| 做多农业 | 豆粕ETF、农业ETF | 能源→化肥→粮价传导已启动 |

**有条件（取决于体制和情景）：**

| 方向 | 标的示例 | 进场条件 | 止损条件 |
|------|---------|---------|---------|
| 做空/减持纳指 | QQQ Put、IC期货 | 体制=滞胀 且 VIX>25 | 停战概率>50%时 |
| 做多航运 | 中远海控(601919)、招商轮船 | 运价持续涨+情景B/C | 停火后运价快速回落 |
| 做多煤炭 | 中国神华、煤炭ETF(515220) | 油价>$100持续 | 国内调控政策 |
| 做空中东ETF | 空KSA/UAE | 沙特被拖入信号出现 | 沙特保持中立 |
| 做多美元/空新兴 | 美元ETF | 体制=美元虹吸 | 停战+美元回落 |

**拐点交易（高赔率需时机）：**

| 方向 | 标的示例 | 进场信号 |
|------|---------|---------|
| 抄底中证500 | 中证500ETF(510500) | 停战概率>70% 且 VIX<20 |
| 做空原油 | 原油Put/反向ETF | 停战概率从<30%跳升至>50% |
| 抄底纳斯达克 | QQQ/纳指ETF | VIX见顶回落 + 美联储暗示降息 |

#### 拐点监控

高亮显示接近触发条件的信号：

| 监控项 | 触发阈值 | 含义 |
|--------|---------|------|
| Polymarket 最高概率 | >70% | 停战交易窗口开启 |
| Polymarket 最高概率 | 单日跳升>10pp | 可能有内部消息泄露 |
| 军工ETF (ITA) | 从高点回落>5% | 聪明钱开始撤退 |
| 伊朗里亚尔 | 单日贬值>5% | 伊朗经济接近崩溃点 |
| VIX | 从高点回落破20 | 恐慌消退，风险偏好恢复 |
| Brent | 跌破$90 | 供应恐慌缓解 |

## API 设计

```
GET  /                              # 看板页面
GET  /api/batch/{1-5}               # 分批获取指标（每批5-6个），5分钟缓存
GET  /api/polymarket                # 获取Polymarket停战预测，5分钟缓存
POST /api/analyze                   # 提交合并数据，返回分析结果并存快照
GET  /api/refresh                   # 一次性获取全部数据（本地开发用），5分钟缓存
GET  /api/latest                    # 获取最近一次缓存数据
GET  /api/history/{symbol}?period=1mo  # 获取某指标历史数据
GET  /api/snapshots?limit=50           # 获取历史快照
```

### 批量分组

27个指标分5个批次，避免 Vercel 10秒超时：

| 批次 | 指标 |
|------|------|
| Batch 1 | VIX, Brent, WTI, 天然气, 黄金, 美元指数 |
| Batch 2 | 10Y美债, S&P500, 道琼斯, 纳斯达克, 铜, 铝 |
| Batch 3 | 小麦, 玉米, 军工ETF, LMT, RTX, NOC |
| Batch 4 | ZIM, BDRY, 伊朗里亚尔, 沙特ETF, 以色列ETF, 阿联酋ETF |
| Batch 5 | 印度VIX, TIP, RINF |

### 5分钟后端缓存机制

- 所有数据获取端点（batch/polymarket/refresh）共享同一缓存逻辑
- 后端检查数据库最近一条快照的时间戳（北京时间）
- **5分钟内**：直接返回缓存数据，不调用 Yahoo Finance / Polymarket，响应头 `X-Cache: HIT`
- **超过5分钟**：正常请求外部数据源，存入新快照，响应头 `X-Cache: MISS`
- 所有用户共享同一缓存，防止多用户同时访问导致数据源限流
- 前端根据 `X-Cache` 响应头显示 "缓存数据 (X分X秒前获取)" 或 "最后更新: 时间"

## 前端布局

看板分为以下板块，自上而下：

1. 核心恐慌指标 (VIX)
2. 能源价格 (Brent/WTI/天然气)
3. 避险资产 (黄金/美元/美债)
4. 美股指数 (标普/道指/纳指)
5. 工业金属 (铜/铝)
6. 粮食价格 (小麦/玉米)
7. 军工/航运板块 (6只个股和ETF)
8. 中东/伊朗 (里亚尔+3个中东ETF)
9. 新兴市场风险 (印度VIX)
10. 通胀预期 (TIP/RINF)
11. Polymarket 停战预测 (条形图)
12. **综合分析面板**（新）
    - 经济体制判定徽章
    - 情景概率仪表盘 (A/B/C 三种情景及概率)
    - 关键信号列表
    - 操作建议表 (确定性方向 + 有条件方向 + 拐点交易)
    - 拐点监控高亮
13. 集运欧线 (待接入)

每个指标卡片包含：当前值、涨跌幅、迷你趋势图、底部小字说明用途。

## 文件结构

```
panelwar/
├── DESIGN.md              # 本设计文档
├── vercel.json            # Vercel 部署配置（builds + routes）
├── requirements.txt       # Python 依赖（Vercel 构建用）
├── .vercelignore          # Vercel 忽略文件

│   ├── index.py           # FastAPI 入口 + 所有API端点（含5分钟缓存）
│   ├── data_fetcher.py    # 数据获取模块（Yahoo Finance, 27个指标, 5批次）
│   ├── polymarket.py      # Polymarket Gamma API 对接
│   ├── analyzer.py        # 分析引擎（四层因果框架，纯规则引擎）
│   └── database.py        # SQLite 数据库（Vercel 用 /tmp，本地用 data/）
├── public/
│   └── index.html         # 单页看板（HTML+CSS+JS 内联）
├── backend/
│   └── main.py            # 本地开发入口（import from api/）
└── data/
    └── panel.db           # SQLite 数据库文件（自动创建）
```
